# Kirby Resolve

*Working on it*

## Installation

### Download

Download and copy this repository to `/site/plugins/resolve`.

### Git submodule

```
git submodule add https://github.com/lukaskleinschmidt/kirby-resolve.git site/plugins/resolve
```

### Composer

```
composer require lukaskleinschmidt/kirby-resolve
```
